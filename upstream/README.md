# Upstream Appendix

This directory contains optional follow-up artifacts for upstream LangSmith CLI and LangChain docs work:

- `langsmith-cli-pr-body.md`: PR body draft for the CLI change.
- `langchain-docs-pr-body.md`: PR body draft for the docs change.
- `patches/`: patch files that demonstrate one possible implementation.

These files are included as review material. The main evidence package does not depend on upstream PRs being opened or accepted.
